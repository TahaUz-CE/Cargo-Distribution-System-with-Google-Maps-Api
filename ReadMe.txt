Things to Consider While Opening the Program(*ENG*)

The -BridgesCargo file is used to open the Java client interface. It is the interface used to record the customer's cargo information.

-MySqlTry1 file is the map part of the project. To run this part, you need to download and install the "GMap.NET.Windows" and "MySql.Data" libraries.
required.

-Change the folder named 'Google Maps Api GUI' to 'MySqlTry1' if the file does not work.

- Since the two GUIs communicate with each other with the database in the project, the database is written in the code by paying attention to 
the names of the tables and variables. Do not forget to determine according to that order.

Programı Açarken Dikkat Edilmesi Gerekenler(*TR*)

-BridgesCargo dosyası Java müşteri arayüzünü açmak için kullanılıyor. Müşterinin kargo bilgilerini kayıt etmek için kullanılan arayüzdür.

-MySqlTry1 dosyası projenin harita kısmıdır. Bu kısımı çalıştırmak için "GMap.NET.Windows" ve "MySql.Data" kütüphanelerini indirip kurmanız
gerekmektedir.

-'Google Maps Api GUI' adlı klasörü eğer dosyanın çalışmaması durumunda 'MySqlTry1' olarak değiştirin.

-Projede iki GUI birbiriyle veritabanı ile birlikte haberleştiği için veritabanını kod da yazıldığı isimlere dikkat ederek tabloları ve değişkenleri
o düzene göre belirlemeyi unutmayınız.
